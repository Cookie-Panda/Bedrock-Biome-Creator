Bedrock Biome Creator is a web tool for Minecraft Bedrock Edition which can create custom biomes based add-ons. It features dynamic JSON generation and mcaddon export.
This tool can create custom biomes behavior pack JSON file, Custom client biome resource pack packed in mcaddon file.
Tou can import, paste your code to customize it.
Updated to latest Beta/preview custom biomes API documentation,
You can create your own project and add multiple biomes in one pack.
multi tab support,
works in offline 
optimized for desktop and mobile devices.
Exported packs can be updated within app i.e. new updates can be exported for pack.
individual json code can be copied and download.
